Welcome to SimBreviloquent

Your go-to plugin for integrating SimBrief with X-Plane.

Features

  • Seamless data download from SimBrief
  • Real-time updates in X-Plane
  • User-friendly interface

Installation

Follow these steps to install the SimBreviloquent plugin:

  1. Download the plugin from the GitHub repository.
  2. Extract the files into your X-Plane plugins folder.
  3. Launch X-Plane and configure the plugin settings.
